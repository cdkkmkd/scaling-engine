# 1分钱AI工具站 🚀

为学生和上班族打造的平价AI工具集合。一个完整的、开箱即用的单页面应用。

## ✨ 核心功能

### 三大AI工具
1. **✍️ AI文案生成** - 输入主题，一键生成营销文案
2. **🎯 AI起名工具** - 为企业/产品生成创意名字
3. **📸 OCR图片文字提取** - 识别图片中的文字（无需后端）

### 财务系统
- **本地次数存储** - 用户次数存储在浏览器，关闭重开也不会丢失
- **三档付费套餐** - 0.01元/1次、0.99元/100次（7天）、9.9元/永久无限
- **权限控制** - 无次数自动弹出付费引导，扣除机制自动化

### 用户体验
- 📱 完全响应式设计 - 电脑/手机完美适配
- 🎨 简洁干净的UI - 适合学生群体
- 🔔 实时通知提示 - 操作即时反馈

## 🚀 快速开始

### 方式1：直接打开
1. 双击或拖拽 `index.html` 到浏览器
2. 立即在本地运行，所有功能正常

### 方式2：本地服务器
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```
然后访问：`http://localhost:8000/index.html`

## 🔧 自定义配置

文件中已标注所有可修改的配置，新手可直接修改：

### 修改套餐价格和次数
```javascript
// 套餐1：改金额、文案、解锁次数
<button onclick="showPaymentModal(1, 0.01, '单次体验', 1)">

// 套餐2：改金额、套餐名、解锁次数
<button onclick="showPaymentModal(2, 0.99, '周卡套餐', 100)">

// 套餐3：改金额、套餐名、解锁次数
<button onclick="showPaymentModal(3, 9.9, '永久会员', 99999)">
```

### 修改AI文案模板
在 `COPYWRITING_TEMPLATES` 数组中修改或添加模板：
```javascript
{
    keyword: '关键词1|关键词2',
    templates: [
        '文案模板1',
        '文案模板2',
        // 添加更多...
    ]
}
```

### 修改AI起名库
在 `NAMING_DATABASE` 对象中修改或添加行业的名字库：
```javascript
'行业名称': {
    examples: ['名字1', '名字2', '名字3']
}
```

### 替换收款码
搜索代码中的 `【收款码替换】` 注释位置，将 `.qr-code-placeholder` 的HTML替换为：
```html
<img src="你的收款码图片URL" alt="收款码">
```

## 📊 数据存储说明

所有用户数据存储在浏览器 `localStorage` 中：
- **totalTimes** - 用户购买的总次数
- **usedTimes** - 已使用的次数
- **vipStatus** - VIP状态（true/false）
- **vipUntil** - VIP过期时间（ISO格式）

清除浏览器缓存会重置数据。

- **适配** - CSS媒体查询响应式设计
